## 🚀 部署建议

### 1. 替换知识库文件
```bash
# 备份旧版本
cp knowledge_base.py knowledge_base.py.bak

# 使用新版本
cp knowledge_base_v4.py knowledge_base.py
```

### 2. 更新 app.py
已自动更新 `app.py` 以导入 `knowledge_base_v4`：
```python
from knowledge_base_v4 import SCENIC_SPOTS, TRAVEL_TIPS, CATEGORY_INDEX, DISTRICT_STATS
```

### 3. 重启服务
```bash
# 停止旧服务（Ctrl+C）
# 启动新服务
python app.py
```

### 4. 验证服务
访问 `http://localhost:5001` 测试：
- 侧边栏应按行政区划分组显示30个景点
- 搜索功能应返回更准确的结果
